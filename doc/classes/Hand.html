active?()

Returns if the player is still able to play in the round or not.

bet=( wager )

Places bet if the bet is valid.

blackjack?()

Determines if current hand is considered blackjack.

busted?()

Determines if hand busted.

clear_bet!()

Sets the current bet to 0

double_down( card )

If the double down is allowed, the bet is doubled, a card is added to Hand, and then stand is called. Otherwise, error is raised.

double_down_allowed?()

A double down is allowed whenever a player has enough money to double their bet.

hit( card )

Hits the player by adding a card to their hand and updating their total. Returns the total.

is_valid_bet?( wager )

Returns whether the bet being placed is valid or not. Valid bets must be an integer value and must be between $1 and the amount of cash that that player has.

lose?( hand )

Determines is the hand loses up against another hand.

push?( hand )

Determines if this hand and the hand it is up against ties.

reset!()

Resets the hand to its original state.

size()

Returns how many cards are in the hand.

soft?()

Indicates if the total is a soft total or not.

split()

If splitting is allowed 2 new hands are created and returned to be placed in Player’s hands, and the current hand is invalidated. Otherwise, error is raised.

split_allowed?()

A split is allowed whenever a player has enough money to double their bet and the hand is Hand#splittable?

splittable?()

A hand is splittable if the player has only two cards, and the two cards have the same face value

stand()

Makes the player inactive, and returns the total.

to_s()

Returns a comma delimited string of all of the cards in the hand.

total()

Gets the total value of the cards in the player’s hand.

win?( hand )

Determines is the hand wins up against another hand.

win_by_blackjack?( hand )

If the hand is considered a win and considered blackjack.

winnings( dealers_hand )

Computes how much money a hand wins, depending on how it was won/lost.
