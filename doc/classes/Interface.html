Module Interface
In: bin/interface.rb

Interface abstracts a lot of the elements of the console interface out into one location.

Methods

clear_screen   header   input   main_menu   newline   pause   pause_and_clear_screen   seperator  

Public Instance methods

clear_screen()

Clears the screen by placing many newline characters into the stream, pushing old text off screen.

header()

Displays the main header for the program.

input( message )

Displays the message passed to the function, and gets a string from the standard input stream. This method takes a block that specifies what is an acceptable string and will loop until it is satisfied.

main_menu()

Displays the main menu for the program and returns their selection as a symbol.

newline()

Simply goes to the next line.

pause()

Displays a message and waits until enter is pressed before continuing

pause_and_clear_screen()

Pauses and clears screen… Imagine that.

seperator()

Displays a simple seperator.
